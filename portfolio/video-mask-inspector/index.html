01. OVERVIEW

Video Mask Inspector

Video Mask Inspector는 비디오 세그멘테이션 데이터 내 객체 고유 식별의 시간적 일관성(Temporal Consistency)을 검수하고 정제하기 위해 개발된 고정밀 GUI 애플리케이션입니다. Meta의 SAM 2.1 모델 추론 엔진과 고성능 뷰포트 렌더링, 증분 파일 동기화 시스템을 탑재하여 대규모 비디오 라벨 데이터 검수 공정을 간소화합니다.

핵심 스펙 표준

  • AI 기반 마스크 편집: SAM 2.1 Tiny, Small, Base+, Large 등 전 규격 모델 탑재
  • 16-bit uint16 규격: 단일 비디오 시퀀스 내 255개 이상의 독립 객체 식별 지원
  • 스마트 증분 동기화: NAS 저장 시 로컬에서 변경된 물리 파일만 추출하여 원격 전송
02. PROBLEM DEFINITION

시간적 ID 불일치 결함

비디오 마스크 전파 시, 객체가 겹치거나 가려지는 현상이 생겼을 때 트래킹 모델이 고유 식별 정보를 유실하여 동일 객체에 다른 ID를 잘못 덮어씌우는 ID Switch 현상이 발생합니다. 이에 따라 전체 시퀀스를 추적하는 인공지능 학습 정답지(Ground Truth)의 무결성이 붕괴됩니다.

데이터 정합성 영향

ID Switch 결함이 잔존하는 비디오 마스크 셋은 학습용 데이터로서 신뢰할 수 없으므로, 프레임들을 전수 검사하고 결함이 발생하는 프레임 이후의 하위 시퀀스들을 일관성 있게 묶어주는 미세 교정 메커니즘이 수반되어야 합니다.

03. KEY TECHNOLOGY

SAM 2.1 리파인먼트 & ID Swap

본 솔루션은 마우스 클릭 입력을 신속히 추론 가이드 피처로 수렴시키는 SAM 2.1 교정 엔진과, 꼬여버린 두 객체 ID 정보를 특정 프레임 시점부터 비디오 끝까지 영구적으로 맞바꾸어 해결하는 ID Swap 제어 기술을 탑재하였습니다.

공격적 마스크 보정

사용자가 입력한 양성(+) 및 음성(-) 포인트의 영향도를 우선 적용하여 픽셀의 오차 범위를 강력하게 제어

ID Swap (Current to End)

ID 불일치 발생 프레임부터 마지막 프레임 범위까지 단 한 번의 단축키로 두 객체 ID를 즉각 교환

04. SYSTEM ARCHITECTURE

논블로킹 스레드 및 Smart Sync

고해상도 이미지 전파 및 AI 연산 도중 GUI가 얼어붙는 현상(Freezing)을 제거하고자 연산 흐름을 백그라운드 스레드로 격리하였으며, 데이터 저장 시 로컬에서 변경된 파일 블록만 선별하는 증분 동기화(Smart Sync)를 적용하였습니다.

리소스 최적화 기능

  • 백그라운드 스레드: AI 연산 및 마스크 전파 동작을 비동기 Worker 스레드로 위임
  • 경량 커맨드 Undo/Redo: 포인트 조작 이력을 논리적으로 관리하여 물리 메모리 오버헤드 0% 실현
  • Smart Sync: 로컬 변경 사항과 리모트 스토리지 파일을 비교하여 변경분만 선별 업로드
05. WEB INTERFACE MODE

FastAPI Web Inspector

Video Mask Inspector는 독립 실행형 PySide6(Qt) 데스크톱 환경 제공에 그치지 않고, FastAPI 백엔드 연동을 구현하여 사용자가 웹 브라우저를 통해 원격으로 마스크 인스펙터 인터페이스에 접속하여 동일한 검수 기능과 조작 단축키를 영위할 수 있도록 보장합니다.

웹 서버 구성 요건

리포지토리 내 backend/main.py 서버 모듈 구동을 통해 localhost:8000 포트로 외부 원격 웹 환경 바인딩을 가동할 수 있으며, 이로써 로컬 장비의 하드웨어 리소스 한계를 우회하여 통합 협업 검수 시스템을 지원합니다.
