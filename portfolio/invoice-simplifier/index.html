02

견적 작성 과정의 기존 비효율성

수동 가격 조회 및 매칭

고객의 체크인과 체크아웃 일자를 확인하여 평일 요금과 주말(금요일, 토요일) 요금을 달력에서 대조하며 일일이 손으로 숙박 일수를 세어 계산하느라 시간이 소요됩니다.

수작업 계산 누수 위험

수동으로 날짜와 단가를 대입하다 보니 주말 일수 미스 계산, 청소비 반영 누락, 부가세 포함/별도 정산 에러 등 단순 계산 실수가 발생하기 쉽습니다.

반복적인 수동 렌더링

매번 엑셀 포맷을 켜서 게스트 정보를 입력하고, 입금 안내 계좌번호를 적어 이미지 캡처 도구로 파일화하여 카카오톡이나 문자 메시지로 발송하는 수공업식 반복 업무가 수반됩니다.

03

견적서 처리 간소화 흐름

01

자연어 예약 정보 구조화

예약 문의 문구를 텔레그램 봇으로 전송하면 Gemini Flash API를 통해 pydantic 데이터 스키마 형식의 JSON 데이터 객체로 정교하게 자동 매핑합니다.

02

요일 요금 자동 연산

바닐라 자바스크립트 Date API를 연동하여 예약 일정 내 금요일과 토요일(주말)을 가려내어 요율대로 자동 가중합 계산을 수행합니다.

03

헤드리스 템플릿 이미지화

임시 HTML에 데이터를 매핑하고, 셀레늄(Selenium) 백그라운드 구동을 통해 html2canvas로 견적서 이미지를 1초 만에 캡처하여 파일로 전송합니다.

Structured JSON Parsing

genai 라이브러리의 구조화된 출력 기능을 설정하여, 자연어 요청문으로부터 신뢰도 높은 게스트 성함, 방 번호, 일정 정보 JSON을 추출하는 단계입니다.

Automatic Day Splitter

체크인부터 체크아웃 기간 동안의 날짜를 일일이 루프 순회하여 숙박 일수와 부가세(incl/excl) 및 청소비 가산 조건을 정확하게 판정해내는 로직입니다.

Automated Screenshot Pipeline

html2canvas 스크립트 실행 제어를 통해 펜션 로고와 계좌번호, 취소 환불 규정이 포함된 정형화된 견적서 이미지를 자동 추출하는 파이프라인입니다.

업무 간소화 효과 및 기술적 비전

예약 파싱 구조화

자연어 요청을 텔레그램 봇으로 수신하여 Gemini Flash API의 pydantic 스키마를 통해 구조화된 JSON 데이터로 파싱합니다.

자동화 렌더링 엔진

headless 크롬 드라이버 기반의 Selenium 백엔드 연동을 통해 통합 HTML 템플릿에 동적으로 데이터를 즉각 바인딩합니다.

데이터 정형성 확보

정제된 JSON 형태의 데이터를 보존하여 데이터 정합성 유지와 향후 타 예약 플랫폼 연동 등 시스템 확장성을 보장합니다.
