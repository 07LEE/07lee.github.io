Nouvel 01

문제 정의

발화자 구분이 필요한 분야

웹소설 작가와의 인터뷰
  1. 한국어를 지원하는 창작 도우미 서비스에 대한 수요
  2. 드라마/영화의 분량에 맞춰 소설을 각색하는 서비스 필요
  3. 각 장면의 구분을 도와주는 서비스 필요
인공지능 솔루션

인공지능으로 발화자 및 사건을 인식, 재가공해 대본의 초석을 제공하는 서비스

콘텐츠 시장의 수요 확대

유튜브 편집 및 오디오북: 기존의 발화자 구분 기술은 반드시 음향 파형 데이터가 존재해야만 기능하며, 순수 텍스트만 주어졌을 때 화자 인식이 불가능해 2차 창작 시 대본화 병목이 발생함

OTT 서비스 자막 도입: 넷플릭스 등 국내외 OTT 시장에서 예능 및 영화에 대사·화자 표시 자막 공급이 확대됨에 따라 텍스트 분석 및 장면 분할 자동화 가치가 대폭 성장함

Nouvel 02

모델 : 파인 튜닝

토크나이저 분석 및 하이퍼파라미터 조건 설정

Tokenizer

1. Input Data

  • 띄어쓰기 반영 여부 검토
  • 형태소 분석 반영 여부 검토
  • 글자 단위 파싱 적용

2. 형태소 분석기

  • KoNLPy (okt, komoran) 튜닝
BERT

1. 크기 및 모델 선택

  • Roberta-klue-large / small
  • polyglot-ko, KoAlpaca 검토

2. 토크나이저

  • 다양한 한국어 토크나이저 적용
  • [UNK] 결측 토큰 제외 설정
ETC

1. 인스턴스

  • Window Size (ws) 크기 튜닝
  • 장면(Scene) 및 시간 가중치 추가

2. 장르

  • 웹소설 장르별 학습셋 구성
  • 장르별 성능 변동 확인

개선해야 할 문제점

  • 하나의 소설만을 기반으로 하여, 새로운 소설에 대한 정확도를 신뢰할 수 없음.
  • 중국어 기반 모델로, 한국어 텍스트에 일부 부적합함.
  • Namelist 생성이 수동으로 이루어짐.
Nouvel 03

결과 분석 및 기대 효과

개발된 발화자 구분 모델의 실무 응용 및 확장성

  • 다자 간 대화가 포함된 모든 서비스에서 활용 가능
    1. 음성 인식에서의 화자 인식 정확도 향상 (텍스트 + 오디오 결합)
    2. 상담 기록 / 인터뷰 내용 등 손쉽게 발화자 구분 가능
    3. 처음 읽는 글에서 내용을 눈에 띄게 구분하기 위해 활용 가능 (철학서적, 삼국지 등..)
  • 웹소설이 아닌 소설 작품의 대본화 (동일한 파이프라인 구조 적용 가능)
  • 기초적 접근 방식이므로 기타 한국어 NLP 과제에서 유용하게 활용될 수 있을 것
